# Formulário de pesquisa - React

Aplicação de formulário desenvolvida com React, com validação de dados e interface moderna.

## Tecnologias
- React
- JavaScript
- CSS3
- Yup
- Formik

## Funcionalidades
- Validação de email
- Inputs organizados em grid
- Interface responsiva
- Navegação entre páginas
- Gerenciar de formulários, tornando o processo de validação mais simples e a manutenção do código mais fácil
